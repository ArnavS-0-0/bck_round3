# AI Hospital Response Suggestion System

An AI-powered clinical response assistant designed for hospital environments. The system enables patients to submit medical queries and generates draft responses using AI, which hospital staff can review, edit, and approve before sending.

## Tech Stack

- **Backend** — FastAPI with Google Gemini API for AI-powered response generation
- **Frontend** — HTML, CSS, and JavaScript UI for patient submission and hospital staff management
